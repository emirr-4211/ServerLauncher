# ServerLauncher

Arkadaş gruplarıyla kolay ve sorunsuz şekilde Minecraft oynamak için geliştirilmiş server tabanlı özel bir launcher.

## Özellikler

* Otomatik sunucu bağlantısı
* Server ile uyumlu mod ve versiyon yönetimi
* Launcher üzerinden güncelleme sistemi
* Hızlı başlatma ve optimize edilmiş performans
* Sorunsuz multiplayer deneyimi

## Nasıl Çalışır

Bu launcher, kullanıcıların manuel kurulum yapmasına gerek kalmadan tüm gerekli dosyaları otomatik olarak indirir ve yapılandırır.
Tüm sistem server ile senkronize çalışır, böylece herkes aynı sürüm ve ayarlarla oyuna girer.

## Kullanım

1. Launcher'ı indir
2. Uygulamayı başlat
3. Giriş yap
4. Play butonuna basarak oyuna gir

## Sistem Yapısı

* Server tabanlı mimari
* Otomatik dosya indirme ve güncelleme
* Versiyon uyumsuzluklarını önleyen yapı

## Notlar

* İlk çalıştırmada gerekli dosyalar indirileceği için açılış süresi uzayabilir
* Stabil kullanım için internet bağlantısı gereklidir

## Amaç

Bu proje, arkadaşlar arasında ortak ve sorunsuz bir oyun ortamı oluşturmak amacıyla geliştirilmiştir.
Kullanıcıların manuel mod ve kurulum işlemleriyle uğraşmasını ortadan kaldırır.
